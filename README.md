### Setting up locally

1. Clone the project and move to folder.

  ```shell
  git clone https://github.com/NutNut17/nut17-notes.git
  cd nut17-notes
  ```

2. Install dependencies.

  ```shell
  npm install
  ```

3. Run dev server.

  ```shell
  hugo server
  ```

4. Finally, visit `http://localhost:1313/` from your browser.
